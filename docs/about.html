Project Description

This project is a single page web application that allows users to explore crime data from the city of St. Paul. Users can view crime incidents on a map, filter incidents by different criteria, and interact with the data through a REST API.

Tools Used

  • Vue.js
  • Leaflet
  • Node.js
  • Express
  • SQLite
  • HTML, CSS, JavaScript

Demo Video

This short demo shows the main functionality of the application, including filtering crime data and interacting with the map.

Interesting Findings

  • Crime incidents are not evenly distributed across neighborhoods.
  • Some neighborhoods consistently have higher crime counts.
  • Certain crime types appear more frequently in specific areas.
  • Filtering by date changes overall crime trends.
  • Mapping incidents makes patterns easier to see than tables alone.
  • Limiting the number of incidents improves performance.
